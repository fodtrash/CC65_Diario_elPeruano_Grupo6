# CC65_Diario_elPeruano_Grupo6
Procesamiento masivo de texto legal usando datos del Diario Oficial El Peruano. Se propone un pipeline de preprocesamiento NLP optimizado con concurrencia en Go (goroutines, Mutex, WaitGroup) para reducir tiempos. Incluye limpieza, normalización, paralelización y modelado en Promela para validar sincronización.
